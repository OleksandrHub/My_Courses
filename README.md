# 🚀 My_Courses: Road to Multi-Stack Engineer

**Мій шлях від Angular Junior до універсального архітектора систем.**

Цей проект - моя персональна база знань, трансформована у цикл практичних курсів. Головна мета: **перестати бути заручником ШІ** та повернути повний контроль над кожним рядком коду. Я вірю, що топовий інженер - це не той, хто копіює промпти, а той, хто розуміє архітектуру, безпеку та інфраструктуру "під капотом".

---

## 🧠 Філософія проекту
1. **No Magic:** Розбираємося, як працюють технології, а не просто "що написати".
2. **Deep Focus:** Від семантики HTML до складних DevOps-пайплайнів.
3. **AI as a Tool, not a Master:** Використовуємо ШІ для прискорення, але проектуємо логіку самостійно.
4. **T-Shaped Growth:** Поглиблюємо знання в Angular та розширюємося в Backend, CyberSec, AI та Ops.

---

## 🗺️ Дорожня карта (Curriculum)

### 🧱 Phase 1: The Foundation (Frontend Deep Dive)
* **HTML5 & Web Standards** (Семантика, Accessibility, SEO).
* **CSS3 Modern Architecture** (Flex, Grid, Animations, Methodologies).
* **JavaScript/TypeScript Hardcore** (Async, Event Loop, Patterns).
* **Angular Mastery** (RxJS, State Management, Performance).

### ⚙️ Phase 2: The Engine (Backend & Data)
* **Node.js & NestJS** (Схожа архітектура з Angular для швидкого старту).
* **SQL & NoSQL** (PostgreSQL, Redis, MongoDB).
* **API Design** (REST, GraphQL, WebSockets).

### ☁️ Phase 3: The Fortress (DevOps & Security)
* **Infrastructure** (Linux, Docker, Kubernetes).
* **Automation** (CI/CD, Terraform).
* **Cybersecurity** (OWASP, Pentesting, Data Encryption).

### 🤖 Phase 4: The Future (AI & ML)
* **AI Engineering** (LLM Integration, Prompt Engineering, AI Agents).
* **MLOps** (Деплой та підтримка моделей).
